# 天气参数说明

## 天气类型编码对照表

| 编码 | 天气类型 | 英文标识 | 辐射特征 |
|-----|---------|---------|---------|
| 0 | 晴天 | clear | DNI占主导，DHI较低，散射比 0.1-0.2 |
| 1 | 多云 | partly_cloudy | DNI波动剧烈，DHI增加，散射比 0.3-0.5 |
| 2 | 阴天 | overcast | DNI接近零，DHI占主导，散射比 0.8-1.0 |
| 3 | 雨天 | rainy | 所有辐射极低，散射比 ≈ 1.0 |
| 4 | 雾天/霾天 | foggy | DNI降低，DHI比例增加，散射比 0.5-0.8 |

## CSV文件列说明

`weather_power_curves.csv` 文件包含以下列：

| 列名 | 说明 |
|-----|------|
| 时间 | 时间戳（小时分辨率） |
| 0 | 晴天功率 (kW) |
| 1 | 多云功率 (kW) |
| 2 | 阴天功率 (kW) |
| 3 | 雨天功率 (kW) |
| 4 | 雾天/霾天功率 (kW) |

## 使用示例

```python
import pandas as pd

# 读取功率曲线数据
df = pd.read_csv('weather_power_curves.csv', index_col='时间', parse_dates=True)

# 获取晴天功率
clear_power = df[0]

# 获取阴天功率
overcast_power = df[2]
```
