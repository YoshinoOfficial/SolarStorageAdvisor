# 光伏发电典型天气分类及辐射参数调整方法

## 一、天气类型分类

光伏发电系统的输出功率主要受太阳辐射影响，根据辐射特征可将天气分为以下几类：

### 1. 晴天（Clear Sky）
- **特征**：无云或少云，大气透明度高
- **辐射特点**：DNI占主导，DHI较低
- **光伏出力**：接近理论最大值，曲线平滑

### 2. 多云（Partly Cloudy）
- **特征**：云量30%-70%，云层移动导致辐射波动
- **辐射特点**：DNI波动剧烈，DHI增加
- **光伏出力**：波动明显，可能出现"云增强"效应

### 3. 阴天（Overcast）
- **特征**：云量>80%，云层较厚
- **辐射特点**：DNI接近零，DHI占主导
- **光伏出力**：仅为晴天的20%-40%

### 4. 雨天（Rainy）
- **特征**：降水天气，云层厚重
- **辐射特点**：DNI和DHI都很低
- **光伏出力**：极低或接近零

### 5. 雾天/霾天（Foggy/Hazy）
- **特征**：大气能见度低，气溶胶含量高
- **辐射特点**：DNI显著降低，DHI比例增加
- **光伏出力**：中等偏低

---

## 二、辐射参数定义及关系

### 参数说明
- **GHI (Global Horizontal Irradiance)**：总水平辐射 [W/m²]
- **DNI (Direct Normal Irradiance)**：法向直射辐射 [W/m²]
- **DHI (Diffuse Horizontal Irradiance)**：水平散射辐射 [W/m²]

### 数学关系
```
GHI = DNI × cos(θz) + DHI
```
其中 θz 为太阳天顶角

### 辐射分量比例指标
- **直射比（Direct Normal Ratio）**：DNI/GHI_clear
- **散射比（Diffuse Fraction）**：DHI/GHI
- 晴天散射比通常 < 0.2
- 阴天散射比接近 1.0

## 三、各天气类型辐射参数调整方法

### 1. 晴天（基准）
使用pvlib的get_clearsky方法直接获取：

```python
import pvlib
import pandas as pd

location = pvlib.location.Location(
    latitude=39.9, 
    longitude=116.4, 
    tz='Asia/Shanghai'
)

times = pd.date_range('2024-06-21 00:00', '2024-06-21 23:59', freq='1min', tz='Asia/Shanghai')
clearsky = location.get_clearsky(times, model='ineichen')

# 返回的DataFrame包含：ghi, dni, dhi
```

**典型参数特征**（正午时刻）：
| 参数 | 范围 |
|------|------|
| GHI | 800-1000 W/m² |
| DNI | 850-950 W/m² |
| DHI | 50-150 W/m² |
| DHI/GHI | 0.1-0.2 |

---

### 2. 多云天气
在晴天基础上，对DNI添加波动和衰减：

```python
import numpy as np

def generate_partly_cloudy(clearsky, cloud_factor=0.7, variability=0.3):
    """
    生成多云天气辐射数据
    
    Parameters:
    - cloud_factor: 云层遮挡因子 (0.5-0.8)
    - variability: 波动幅度 (0.2-0.4)
    """
    n = len(clearsky)
    
    # 生成随机波动序列
    np.random.seed(42)
    noise = np.random.randn(n)
    # 低通滤波模拟云的移动
    window = 30  # 30分钟窗口
    smoothed_noise = np.convolve(noise, np.ones(window)/window, mode='same')
    
    # DNI波动衰减
    dni_factor = cloud_factor + variability * smoothed_noise
    dni_factor = np.clip(dni_factor, 0.2, 1.0)
    
    cloudy = clearsky.copy()
    cloudy['dni'] = clearsky['dni'] * dni_factor
    
    # DHI增加（云散射效应）
    dhi_enhancement = 1.5 - 0.5 * dni_factor
    cloudy['dhi'] = clearsky['dhi'] * dhi_enhancement
    
    # 重新计算GHI
    solar_position = location.get_solarposition(times)
    cloudy['ghi'] = cloudy['dni'] * np.cos(np.radians(solar_position['zenith'])) + cloudy['dhi']
    cloudy['ghi'] = cloudy['ghi'].clip(lower=0)
    
    return cloudy
```

**典型参数特征**（正午时刻）：
| 参数 | 范围 |
|------|------|
| GHI | 400-700 W/m²（波动） |
| DNI | 300-700 W/m²（波动） |
| DHI | 150-300 W/m² |
| DHI/GHI | 0.3-0.5 |

---

### 3. 阴天
大幅降低DNI，提高DHI比例：

```python
def generate_overcast(clearsky, thickness_factor=0.3):
    """
    生成阴天辐射数据
    
    Parameters:
    - thickness_factor: 云层厚度因子 (0.1-0.4)
    """
    overcast = clearsky.copy()
    
    # DNI几乎为零
    overcast['dni'] = clearsky['dni'] * 0.05
    
    # DHI降低但比例增加
    overcast['dhi'] = clearsky['ghi'] * thickness_factor
    
    # 重新计算GHI
    solar_position = location.get_solarposition(times)
    overcast['ghi'] = overcast['dni'] * np.cos(np.radians(solar_position['zenith'])) + overcast['dhi']
    overcast['ghi'] = overcast['ghi'].clip(lower=0)
    
    return overcast
```

**典型参数特征**（正午时刻）：
| 参数 | 范围 |
|------|------|
| GHI | 100-300 W/m² |
| DNI | 0-50 W/m² |
| DHI | 100-280 W/m² |
| DHI/GHI | 0.8-1.0 |

---

### 4. 雨天
所有辐射参数大幅降低：

```python
def generate_rainy(clearsky, intensity_factor=0.1):
    """
    生成雨天辐射数据
    
    Parameters:
    - intensity_factor: 降雨强度因子 (0.05-0.2)
    """
    rainy = clearsky.copy()
    
    # DNI接近零
    rainy['dni'] = clearsky['dni'] * 0.02
    
    # DHI很低
    rainy['dhi'] = clearsky['ghi'] * intensity_factor
    
    # 重新计算GHI
    solar_position = location.get_solarposition(times)
    rainy['ghi'] = rainy['dni'] * np.cos(np.radians(solar_position['zenith'])) + rainy['dhi']
    rainy['ghi'] = rainy['ghi'].clip(lower=0)
    
    return rainy
```

**典型参数特征**（正午时刻）：
| 参数 | 范围 |
|------|------|
| GHI | 30-100 W/m² |
| DNI | 0-20 W/m² |
| DHI | 30-90 W/m² |
| DHI/GHI | ≈ 1.0 |

---

### 5. 雾天/霾天
DNI显著降低，DHI比例增加：

```python
def generate_foggy(clearsky, visibility_factor=0.4):
    """
    生成雾天/霾天辐射数据
    
    Parameters:
    - visibility_factor: 能见度因子 (0.2-0.5)
    """
    foggy = clearsky.copy()
    
    # DNI显著降低
    foggy['dni'] = clearsky['dni'] * visibility_factor * 0.5
    
    # DHI散射增强
    foggy['dhi'] = clearsky['ghi'] * visibility_factor * 0.8
    
    # 重新计算GHI
    solar_position = location.get_solarposition(times)
    foggy['ghi'] = foggy['dni'] * np.cos(np.radians(solar_position['zenith'])) + foggy['dhi']
    foggy['ghi'] = foggy['ghi'].clip(lower=0)
    
    return foggy
```

**典型参数特征**（正午时刻）：
| 参数 | 范围 |
|------|------|
| GHI | 200-400 W/m² |
| DNI | 100-250 W/m² |
| DHI | 150-300 W/m² |
| DHI/GHI | 0.5-0.8 |

## 五、实际应用建议

### 1. 典型日生成策略
- 每种天气类型生成1-2个典型日
- 考虑季节变化（夏季和冬季晴天辐射差异）
- 多云天气需要考虑时间序列的连续性

### 2. 验证方法
```python
# 检查辐射参数一致性
def validate_irradiance(df):
    solar_position = location.get_solarposition(df.index)
    calculated_ghi = df['dni'] * np.cos(np.radians(solar_position['zenith'])) + df['dhi']
    
    # 允许5%误差
    diff = np.abs(calculated_ghi - df['ghi']) / df['ghi']
    assert (diff < 0.05).mean() > 0.95, "辐射参数不一致"
```

### 3. 与实测数据对比
建议使用当地气象站或卫星数据校准调整系数，不同地区的气溶胶、水汽含量会影响实际辐射值。
